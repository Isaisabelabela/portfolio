Tecnologias Utilizadas

Resumo e Análise de Sentimentos com IA

Aplicativo web para resumir textos e identificar o sentimento (positivo ou negativo), utilizando modelos pré-treinados da Hugging Face em uma interface simples com Streamlit.

Acessar Repositório

Tecnologias Utilizadas

  • Python: linguagem principal do projeto
  • Streamlit: criação da interface web interativa
  • Transformers (Hugging Face): uso de modelos para NLP
  • pipeline: - summarization com modelo sshleifer/distilbart-cnn-12-6
    - sentiment-analysis com modelo distilbert-base-uncased-finetuned-sst-2-english
  • dotenv: leitura segura de variáveis de ambiente (API key)
