Prazer, Isabela!

Sou estudante de Ciência da Computação e apaixonada por transformar dados brutos em ideias que realmente fazem sentido. Gosto de olhar para planilhas e enxergar oportunidades — e quando não estou fazendo isso, provavelmente estou dando vida a alguma interface com código com café do lado. ✨

No dia a dia, uso Python e SQL para analisar dados, automatizar tarefas e, as vezes, corrigir os bugs que eu mesma criei 😅. Também me divirto no front-end, criando páginas com HTML, CSS e um toque de PHP, sempre tentando deixar tudo mais leve, bonito e funcional.

Gosto de resolver problemas na base da lógica e aprender testando na prática. Ah, e pra sair um pouco do mundo dos códigos, nada melhor do que um bom treino ou uma caminhada ao ar livre — é assim que recarrego as energias!

Habilidades em destaque:

  • Manipulação de dados com Python (Pandas, NumPy, etc.)
  • Consultas, joins e modelagem com SQL
  • Front-end com HTML, CSS e boas práticas visuais
  • Desenvolvimento com PHP
