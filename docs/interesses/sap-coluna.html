Como adicionar uma coluna em uma tabela no SAP (SE11)

Este guia mostra como adicionar um novo campo (coluna) em uma tabela existente no SAP, utilizando a transação SE11 (ABAP Dictionary).

O que é uma coluna no SAP?

No SAP, uma coluna corresponde a um campo (field) da tabela. Adicionar uma coluna significa incluir um novo campo na estrutura da tabela, o que altera a definição da base de dados.

Passo a passo para adicionar uma nova coluna

  1. Acesse a transação SE11:
    • Na barra de comandos SAP, digite SE11 e pressione Enter.
    • Você será direcionado ao ABAP Dictionary.
  2. Selecione a opção Tabela e insira o nome da tabela:
    • Exemplo: ZMEU_EXEMPLO_TB
    • Clique em Exibir (Display) ou Alterar (Change), se tiver permissão.
  3. Clique na aba ou botão Campos (Fields):
    • Você verá a estrutura da tabela com todos os campos existentes.
  4. Adicione uma nova linha com os seguintes dados:
Campo Descrição
Nome do campo Nome técnico do novo campo
Tipo de dados Ex: CHAR, NUM, DEC etc.
Comprimento Exemplo: 10 (para CHAR), 13.2 (para DEC com 2 casas decimais)
Descrição Nome explicativo do campo

Dica: use nomes em letras maiúsculas e com prefixos claros que identifiquem o campo.

  1. Salve e ative a tabela:
    • Clique no ícone de disquete para salvar.
    • Clique no ícone de martelo para ativar a tabela.
    • Se estiver em ambiente de desenvolvimento, o SAP pode solicitar uma ordem de transporte.

O que acontece após adicionar um campo?

  • O novo campo passa a fazer parte da base de dados da tabela.
  • Ele pode ser usado em relatórios, ALVs, formulários, integrações e programas ABAP.

Importante

  • Você só pode alterar tabelas Z (customizadas). Tabelas padrão da SAP não podem ser modificadas diretamente.
  • Se a tabela já possui registros, o novo campo será adicionado com valor NULL até que seja preenchido manualmente ou via código ABAP.

Exemplo prático

Suponha que deseja adicionar a coluna COD_CATEGORIA com tipo CHAR e tamanho 4. O preenchimento seria:

Campo Tipo Comprimento Texto explicativo
COD_CATEGORIA CHAR 4 Código da categoria

Resumo ilustrado

A sequência básica para adicionar um campo é:

SE11 → Digite o nome da tabela → Clique em ALTERAR → Vá na aba CAMPOS → Adicione o novo campo → SALVE e ATIVE → PRONTO
