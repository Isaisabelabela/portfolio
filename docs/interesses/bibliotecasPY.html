Ferramentas que uso em projetos com Python

Essas são as bibliotecas que tornam meu processo de análise e modelagem de dados mais simples, divertido e poderoso.

Python — o cérebro da operação

Simples e versátil, Python comanda tudo no projeto: leitura de dados, processamento, visualização e modelagem. É o ponto de partida e também o elo entre todas as outras ferramentas. 

import pandas as pd
df = pd.read_csv('vendas.csv')
print(df.head())

Pandas e NumPy — dupla dinâmica

Pandas organiza dados como tabelas. NumPy faz contas e cálculos eficientes. São essenciais para transformar dados brutos em informações úteis. 

# Pandas
vendas_altas = df[df['valor'] > 1000]

# NumPy
import numpy as np
media = np.mean([10, 20, 30])

Matplotlib e Seaborn — visual bonito e claro

Com Matplotlib você tem o básico da visualização. Já o Seaborn dá um toque de design e torna tudo mais interpretável com menos código. 

import seaborn as sns
sns.countplot(data=df, x='produto')

Plotly — gráficos interativos

Gráficos que respondem ao mouse e criam dashboards incríveis. O Plotly é ideal para transformar relatórios estáticos em experiências dinâmicas. 

import plotly.express as px
fig = px.pie(df, names='categoria', values='vendas')
fig.show()

scikit-learn — machine learning sem dor

Quer treinar um modelo de regressão ou classificação com poucas linhas? O scikit-learn é sua melhor escolha, com APIs simples e diretas. 

from sklearn.linear_model import LinearRegression
modelo = LinearRegression()
modelo.fit(X_train, y_train)

XGBoost — o motor turbo do aprendizado

Quando você quer performance em modelos preditivos, o XGBoost entra em cena com boosting poderoso e otimizações inteligentes. 

import xgboost as xgb
modelo = xgb.XGBRegressor()
modelo.fit(X_train, y_train)

Faker — dados fictícios com estilo real

Precisa simular um banco de dados para teste? O Faker gera nomes, emails, telefones e muito mais — perfeito para testes sem violar privacidade. 

from faker import Faker
fake = Faker()
print(fake.name(), '-', fake.email())

Resumo visual

Essas ferramentas me ajudam a construir desde a base até o topo de um projeto de dados completo: coleta, análise, visualização, aprendizado de máquina e apresentação.
